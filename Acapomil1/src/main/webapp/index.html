EJERCICIOS EN CLASES

1. Crear un formulario que solicite el nombre y la edad de una persona. Al enviar el formulario, mostrar un mensaje que diga "Hola, (nombre), tienes (edad) años".

2. Crear un formulario que solicite el nombre y la edad de una persona. Al enviar el formulario, mostrar un mensaje que diga "Hola, (nombre), tienes (edad) años". Si la edad es mayor de 18 años, mostrar un mensaje adicional que diga "Eres mayor de edad".

3. Crear un formulario que solicite el nombre y la edad de una persona. Al enviar el formulario, mostrar un mensaje que diga "Hola, (nombre), tienes (edad) años". Si la edad es mayor de 18 años, mostrar un mensaje adicional que diga "Eres mayor de edad". Si la edad es menor de 18 años, mostrar un mensaje adicional que diga "Eres menor de edad".
